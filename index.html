คำจำกัดความ

วิ่งตามภูมิประเทศ

กีฬาที่อาศัยความทนทาน วิ่งบนเส้นทางธรรมชาติ — ป่า ภูเขา ทางลูกรัง — มีการเดิน-วิ่งปนกัน ต้องใช้สมรรถนะทุกด้านทั้งความเร็ว ความทน และพลังกล้ามเนื้อ

≠ Cross-Country ≠ วิ่งถนน ≥8 กม. = มีความเสี่ยง

หลักการสำคัญ

3 สิ่งต้องระลึกเสมอ

  • 📍 ทุกพื้นที่มีเจ้าของ — ต้องขออนุญาตก่อนเสมอ
  • 🌿 ต้องไม่สร้างความเสียหายต่อสิ่งแวดล้อม
  • 🛡️ ผู้จัดคือผู้รับผิดชอบเพียงผู้เดียว — ปัดไม่ได้
ประเภทระยะทาง

3 ประเภท — ความต้องการต่างกัน

แต่ละระยะมีข้อกำหนดป้ายและการจัดการที่แตกต่างกัน

ระยะสั้น

< 16 กม.
  • วิ่งเร็ว — ดูแผนที่ไม่ทัน
  • ต้องมีป้ายบอกทาง ชัดมากเป็นพิเศษ
  • มาร์แชลทุกจุดเสี่ยง
  • ทดสอบป้ายโดยการวิ่งจริง

ระยะกลาง

16 – 32 กม.
  • ใช้แผนที่ประกอบได้บ้าง
  • ต้องมีป้ายทุกจุดสับสน
  • วางแผนการติดป้าย 1 วัน/32 กม.
  • มาร์แชลประจำจุดเสี่ยง

ระยะไกล

> 32 กม.
  • วิ่งช้า — อ่านแผนที่ได้
  • แนะนำให้ Download GPS track
  • ป้ายสะท้อนแสง (สำหรับวิ่งกลางคืน)
  • ต้องมีแผนการแพทย์ครอบคลุม
Timeline การจัดงาน

5 ขั้นตอนหลัก — ตั้งแต่ต้นจนจบ

ITRA International Standard

การประเมินสนาม — มาตรฐานสากล

International Trail Running Association ใช้ 3 ตัวชี้วัดในการจัดระดับสนาม

💪

Endurance Points

0–6

บ่งบอกความยากโดยรวมของสนาม (ระยะทาง + ความสูงสะสม)

⛰️

Mountain Level

0–12

ความยากด้านภูมิประเทศและความสูง สำหรับผู้เข้าร่วมที่ต้องใช้ความพยายามพิเศษ

🏅

Finisher Criteria

%

วิเคราะห์จากข้อมูลสนามและเวลาที่กำหนด คาดการณ์อัตราผู้เข้าเส้นชัย

🗺️

ข้อกำหนดแผนที่

  • → สเกล 1:50,000 ขึ้นไป
  • → รองรับ GPS + พิกัด Grid
  • → ทุกคนได้รับแผนที่ฉบับเดียวกัน
  • → มีคำอธิบายสัญลักษณ์ครบ
📍

แผนที่ต้องแสดง

  • → เส้นทางหลัก + เส้นทางฉุกเฉิน
  • → จุด Checkpoint + Cut-off Time
  • → Aid Stations + สถานีการแพทย์
  • → จุดเสี่ยงภัยและเขตอันตราย
🔴

ระบายสีแผนที่

  • → เส้นทางหลัก (สีเด่น)
  • → เส้นทางฉุกเฉิน (สีอื่น)
  • → เขตอันตราย (สีแดง)
  • → ใช้เฉดสีเพื่ออ่านง่ายขึ้น
🏷️

คำบรรยายเส้นทาง

  • → ตัวอักษร 11 มม. (พกพาสะดวก)
  • → ใส่เลขบรรทัดกำกับ
  • → พิกัดทุก Checkpoint
  • → ระยะทางจากจุดเริ่มต้นทุกจุด
อุปกรณ์บังคับ (Mandatory Gear)

ทุกชิ้นต้องพกติดตัวตลอดการแข่งขัน

🏥 สถานีการแพทย์หลัก (Primary)

  • ตั้งบริเวณที่นักกีฬาผ่านมาก (Aid Station หลัก)
  • พื้นที่ 15 ตร.ม. ต่อนักกีฬา 1,000 คน
  • มีแอร์ (ร้อน) หรือเครื่องทำความร้อน (หนาว)
  • ไฟฟ้า + น้ำสะอาด + โทรศัพท์
  • AED พร้อมใช้งานตลอดเวลา

⛑️ สถานีการแพทย์รอง (Secondary)

  • ตั้งระหว่างสถานีหลัก บนจุดเสี่ยง
  • ทีมช่วยเหลืออย่างน้อย 6 คน (เขตเดิน)
  • อุปกรณ์รักษาอุณหภูมิร่างกาย (ผ้าห่มฉุกเฉิน)
  • พร้อมขนส่งผู้บาดเจ็บออกจากพื้นที่
  • ประสานกับสถานีหลักตลอดเวลา
